# Type Alias: Level

> **Level**: `"silent"` \| `"error"` \| `"warn"` \| `"access"` \| `"info"` \| `"debug"`

Defines the logging level. The level is in order of priority, Debug being
the lowest and Silent being the highest. If Silent is chosen, logs should
never be sent.
